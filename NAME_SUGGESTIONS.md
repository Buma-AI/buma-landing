# Suggestions de Noms pour BumaAI

## Catégorie 1 : Confiance & Progrès (Émotionnel)

### 1. **Confiance+** / **ConfiancePlus**

- **Domaine** : confianceplus.com, confianceplus.fr
- **Pourquoi** : Évoque directement la confiance retrouvée, le "+" suggère l'amélioration
- **Ton** : Rassurant pour parents, motivant pour élèves

### 2. **Éclaire** / **Eclaire**

- **Domaine** : eclaire.app, eclaire-education.com
- **Pourquoi** : "Éclairer le chemin", méthode socratique, lumière sur les difficultés
- **Ton** : Poétique, éducatif, bienveillant

### 3. **Lumi** / **LumiApp**

- **Domaine** : lumi.app, lumi-education.com
- **Pourquoi** : Lumière, clarté, compréhension. Court et mémorable
- **Ton** : Moderne, accessible, positif

### 4. **Élan** / **ElanApp**

- **Domaine** : elan.app, elan-education.com
- **Pourquoi** : Élan vers le progrès, dynamisme, motivation
- **Ton** : Énergique, positif, encourageant
